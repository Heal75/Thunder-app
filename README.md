## Overview
Thunder APP is a personal media player application designed for Android/iOS. It allows users to manage and organize their own local video libraries and personal playlists.

The application focuses on providing a clean, metadata-rich interface for users to browse their legally owned content.

## Key Features
* **Playlist Management:** Users can input their own M3U playlists.
* **Metadata Integration:** Automatically fetches posters, plot summaries, and cast details for the user's content.
* **Clean Interface:** A modern, user-friendly video playback experience.

## Content Disclaimer
This application is a **media player only**. It does not come pre-loaded with any content, movies, or video streams. Users are responsible for providing their own content. The developer does not endorse or support the use of this application for streaming illegal content.
